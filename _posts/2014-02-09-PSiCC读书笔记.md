又是英文版，头痛。PSiCC的全称是 *Practical Statecharts in C/C++:Quantum Programming for Embedded Systems*。 把这本书啃完，状态机就差不多了吧。

###导论
所有的计算机系统都是事件驱动的，不管是通用的还是特殊的嵌入式系统，这就意味着它们在不断的等待事件的发生。就像这样：

等待事件发生-------------->识别事件类型------------>处理事件------->处理完成
    ^                                                                  |
    |                                                                  |
    +------------------------------------------------------------------+
